# Windows Store - Key App Features

Here is a summarized list of the top features for the **IT Quick Runbooks** store listing:

* **Efficient Runbook Execution**: Designed for high-performance operations and SRE teams.
* **Markdown & JSON Support**: Define and manage your documentation using standard, portable formats.
* **Pro Syntax Highlighting**: Integrated PrismJS support for BASH, SQL, YAML, JSON, PowerShell, and more.

* **Contextual Tag Cloud**: Dynamic filtering that updates based on the active service category (IAAS/PAAS).
* **Built-in Runbook Editor**: Create, modify, and preview your procedures without leaving the app.
* **Advanced Source Management**: Load runbooks from local directories or directly from Git repositories.
* **Git Integration**: Support for shallow cloning and easy "Refresh Sources" for synced teams.
* **Premium Media Handling**: Standardized image thumbnails with hover effects and professional lightboxes.
* **Smart Search**: Instantly find any command or procedure across your entire runbook library.
* **Expected Results Tracking**: Every step includes clear "Expected Result" blocks for error-proof execution.
* **Customizable Visualization**: Choose between Super Compact, Compact, Comfortable, or Detailed list modes.
* **GUI Scaling**: Optimized for high-density screens with adjustable interface scaling.
* **Automatic Updates**: Stay up-to-date with integrated background updates and version checking.
* **Template System**: Download standardized Markdown and JSON templates to jumpstart new procedures.
* **Multi-Service Metadata**: Organize by Service (IAAS/PAAS), Category (Database/Network), and Tags.
* **One-Click Copy**: Minimized "Copy" buttons on every code block for lightning-fast command execution.
* **Dark Mode Native**: A premium, high-contrast dark aesthetic designed for long focus sessions.
* **Windows Protocol Support**: Robust `qrun-asset://` protocol for reliable local file and image loading.
* **Portable & Installer Modes**: Available as a standard Windows installer or a zero-install portable executable.
